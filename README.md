#  Analytics Projects

## Overview
This repository contains various business analytics projects aimed at providing insights into different domains. The projects utilize data analysis techniques to uncover patterns and trends.

## Tools & Technologies
- **Microsoft Excel**: Data cleaning and analysis
- **Python**: Programming language for data analysis
- **Jupyter Notebook**: Interactive development environment

## Structure
- `Bike Project.xlsx`: Analysis of bike sales and usage data
- `Ecommerce.xlsx`: Insights into e-commerce sales and customer behavior
- `HR RepositoryAnalytics.xlsx`: Human resources data analysis for employee retention

## Key Insights
- Analyzed sales trends and customer preferences
- Identified factors influencing employee retention
- Provided recommendations for business improvements
